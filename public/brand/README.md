# Brand assets

Drop the official Evntiqa logo files here:

- `logo-light.svg` — used on dark backgrounds (header over the navy hero, footer).
- `logo-dark.svg` — used on light backgrounds (header after scroll, mobile menu).

The current files are clean text placeholders. SVG is preferred for crispness at every size, but PNG works too — if you ship PNGs, update the references in `src/components/Header.tsx` and `src/components/Footer.tsx`.

Recommended dimensions for SVG: ~200×40 (a wide horizontal aspect, like a typical wordmark).
