# Open Graph image

Drop a 1200×630 PNG named `og-image.png` here.
It is referenced by the Open Graph + Twitter Card meta tags in `index.html`
and is what social platforms (LinkedIn, X, Slack, etc.) preview when the URL
is shared. Recommended: brand-blue background, Evntiqa wordmark, the tagline
"ABM-Powered Executive Circles for B2B Tech".
